Repositório dedicado ao estudo de Programação Orientada a Objetos (POO) em Java.

Este projeto contém uma coleção de exercícios e pequenos projetos práticos organizados 
por conceitos fundamentais da POO, como:

Fundamentos da linguagem
Encapsulamento
Herança
Polimorfismo
Abstração
Interfaces

O objetivo é consolidar o aprendizado através da prática, simulando cenários reais 
e aplicando boas práticas de desenvolvimento.

Cada atividade é estruturada de forma progressiva, permitindo evolução contínua no 
domínio da linguagem Java e dos princípios de orientação a objetos.
